Usage Guidelines

Last updated: August 2025

Get the best results from Herodotus with these tips and best practices.

Document Requirements

Supported Formats

  • Text files (.txt) - Plain text documents
  • Word documents (.doc, .docx) - Microsoft Word files
  • Rich Text (.rtf) - Rich Text Format files
  • PDF files (.pdf) - Portable Document Format
Best Results: Include at least 500-1000 words total across all documents for accurate style analysis.

Writing Sample Quality

What Works Best

  • Consistent authorship - All samples from the same writer
  • Similar content type - Emails, articles, or creative writing
  • Recent writing - Reflects current writing style
  • Complete thoughts - Full paragraphs and sentences

What to Avoid

  • Mixed authorship from multiple writers
  • Heavily edited or collaborative content
  • Technical documentation or code
  • Lists, bullet points, or fragmented text

Optimal Usage Patterns

Creating Styles

Recommended: 3-5 documents of 200-500 words each, or 2-3 longer pieces of 500+ words.

Content Generation

  • Clear prompts - Be specific about what you want
  • Context matters - Include relevant background information
  • Length guidance - Specify desired length (paragraph, page, etc.)

Privacy and Security

  • Your documents are processed securely and not shared
  • Remove sensitive information before uploading
  • Generated content reflects your input style and prompts
  • You can delete styles and documents at any time

Performance Tips

  • File size: Keep individual files under 5MB for faster processing
  • Style management: Organize styles by purpose (emails, articles, etc.)
  • Regular updates: Add new documents to keep styles current

Troubleshooting

Poor Style Matching

  • Add more diverse writing samples
  • Ensure samples are from consistent author
  • Include longer-form content

Upload Issues

  • Check file format is supported
  • Ensure file size is under 10MB
  • Try converting complex PDFs to text
